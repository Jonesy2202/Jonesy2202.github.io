My Projects

Below are my projects that I have worked on in my time at university as well as some that I work on in my own time.

My Current Project: Fake Shop (Fausse Boutique)

Fausse Boutique is a fake e-commerce store that uses a fake online store API called FakeStoreAPI that has all of the data needed to make an e-commerce site with products, users and baskets.

This project is still a work in progress so there isn't much to show but I have taken some of my inspiration from some high end shopping sites to give it a professional look.

My Major Project

My major project was a meal planner site that allowed the user to create, view and save recipes that them or other users created. The front-end was coded in HTML, CSS and used Bootstrap for the style and layout of the site and JavaScript with some jQuery for displaying, formatting, retrieving and sending data. The back-end used a PostgreSQL Database and PHP to access and manipulate the data in the database.

The inspiration for this project was to be able to share recipes with family and friends as well as planning meals over a week to help with budgeting and meal prepping because I started to enjoy cooking while at university.

This is the login page for my major project. It also has the register form next to it to avoid using a two separate pages for them both.

Multiplayer Quiz Game

Once I graduated I decided to make some smaller projects to further improve my skills with front-end design and learning to use more APIs. I used Open Trivia Database to learn how to use Postman and how I can use APIs in other projects as this site doesn't require any API key to make requests.

Like my major project, this site uses Bootstrap for the responsive design of the site as it provides a 12 column layout and some pre-made items like cards and buttons to make my development process quicker and easier.

Workout App

In my final year of university I picked an Android development module and I was tasked with creating a workout app to plan workout sessions. The app would allow users to add, remove and edit exercises as well as add them to a session. Some workouts, if applicable, could have a drop set mode turned on for more variety with that exercise. I used Jetpack Compose and Material 3 Design functions to create the app.
